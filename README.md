# 🚖 Taxi Sales Analysis Project

## 📌 Overview
This project is a **Power BI Dashboard** built to analyze taxi booking data.  
It explains business performance in **simple visuals and KPIs** so that anyone can understand:  
- How much revenue is generated  
- Why rides are cancelled  
- How drivers and customers perform  
- What trends exist over time  

The goal is to make taxi business insights **clear, simple, and actionable**.

## 🔑 Key Metrics
- **Service Quality Score**  
- **Revenue Momentum**  
- **Customer Retain Rate**  
- **Premium Peak Hour Index**  
- **Weekly & Monthly Bookings**  
- **Quarterly Avg Booking Value**

---

## 🎨 Features
- Interactive slicers (Date, Customer Segment, Driver Category, City/Zone)  
- Professional UI design with homepage navigation  
- KPIs highlighted via cards and charts  
- Trend analysis with line charts, bar charts, scatter plots, and combo visuals  

---

## 🛠 Tools Used
- Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Data Modeling  

---

## 🚀 How to Use
1. Download the `.pbix` file from this repository.  
2. Open it in **Power BI Desktop**.  
3. Explore dashboards and interact with slicers to analyze different perspectives.  

---

## 📌 Project Summary
This project shows how **data-driven insights** can improve taxi business operations.  
By combining **customer, driver, financial, and time-based KPIs**, the dashboard becomes a **strategic cockpit** for executives to monitor performance and make better decisions.
